# 淼淼 · Portfolio

淼淼（Miaomiao）的个人作品集网站 —— 一名 AI 驱动的独立开发者，应届毕业生。

## 收录作品
- 📱 纯净跳过 PureSkip — 自动跳过开屏广告的安卓工具
- 📊 智能选股分析系统 — 每天自动运行的全栈选股分析系统

## 设计风格
- 暖纸 × 手作墨迹（Warm Paper & Ink）：明亮米白纸感、去饱和赭红主色、手写体与手绘元素
- 交互：滚动揭示、顶部进度条、磁吸按钮、卡片视差倾斜、工具箱跑马灯
- 中英双语一键切换，尊重 `prefers-reduced-motion`

## 技术栈
- HTML5 + CSS3 + 原生 JavaScript（无框架、无构建）
- 响应式设计（移动端优先兼容）
- 部署于 Cloudflare Pages
